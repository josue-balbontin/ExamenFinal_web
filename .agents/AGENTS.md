# Reglas de Comportamiento del Agente

- No usar el modo de planificación ni solicitar confirmación (RequestFeedback: true) a menos que sea algo destructivo irreversible o ambiguo.
- Ejecutar y aplicar los cambios solicitados por el usuario de forma directa y automática, asumiendo la aprobación, pero siguiendo siempre estrictamente las indicaciones dadas.
